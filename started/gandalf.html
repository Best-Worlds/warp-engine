Mode Gandalf

you can set parameters by command-line directly without wizard.

Example:

warp init --mode-gandalf \
    --namespace=howard  \
    --project=gandalf  \
    --framework=m2  \
    --vhost=local.magento2.com  \
    --php=7.2-fpm  \
    --mysql=5.7  \
    --postgres=9.6.15  \
    --elasticsearch=5.6.8  \
    --redis=3.2.10-alpine  \
    --rabbit=3.7-management  \
    --mailhog  \
    --varnish=5.2.1
